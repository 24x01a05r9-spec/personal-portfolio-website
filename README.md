# Personal Portfolio Website

A full-stack personal portfolio website built using Flask, HTML, and CSS. This project showcases personal skills, projects, and contact information with a responsive user interface.

---

## Features

- Home page with personal introduction
- Skills section
- Projects section
- Contact information
- Responsive design
- Flask backend integration

---

## Technologies Used

- Python
- Flask
- HTML
- CSS
- Google Colab

---

## Project Structure

personal-portfolio-website/

├── app.py

├── templates/

│ └── index.html

├── static/

│ └── style.css

└── README.md

---

## Installation

Install required packages:

pip install flask flask-ngrok

---

## Run the Project

Run the Flask application:

python app.py

---

## Output

The application generates a public ngrok URL where the portfolio website can be viewed in the browser.
